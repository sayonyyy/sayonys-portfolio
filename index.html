My Projects

API Testing & Documentation

This project focuses on API testing using the Reqres mock API. It includes comprehensive documentation of both GET and POST request methods. The GET request retrieves a paginated list of users, while the POST request allows the creation of a new user by submitting their name and job. Tools like Postman were used to simulate and test these endpoints, ensuring accurate and reliable API behavior. This documentation demonstrates my ability to understand RESTful APIs, use testing tools, and present technical details clearly.

Music Client Portfolio

The site also includes a clean discography listing, a built-in contact form for booking inquiries, and prominent social media links to help fans connect and follow updates. Kartella

Kartella is a modern edtech and e-commerce platform that empowers students to turn learning into real-world opportunities. Offering certified courses across academic and creative fields — from baking and music to nail art and design — it blends education with entrepreneurship. With built-in e-commerce features and exclusive student benefits, Kartella enables users to showcase, sell, and monetize their skills. FitTech

FitTech is a fitness-focused web app developed during our first hackathon — Smart India Hackathon (SIH) — where our team successfully qualified for the second round. It was also my second project overall, marking a key milestone in my developer journey. FitTech allows users to input their height, weight, and age to receive personalized diet plans, workouts, and sport-specific routines, along with guided fitness videos. Upcoming features include daily progress tracking and an integrated store for purchasing diet foods, supplements, and sports accessories — ensuring users never have to pause their fitness journey due to a lack of resources. FitTech aims to be a one-stop solution for fitness, nutrition, and training, even offering health-based personalization through prescriptions.
