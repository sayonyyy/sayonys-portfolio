My Projects

Titanic Survival Prediction (ML & DL)

Built a machine learning and deep learning application to predict Titanic passenger survival using Scikit-learn (Decision Tree) and PyTorch (Neural Network). Users can input passenger details and get survival predictions via a Streamlit interface. The system also tracks and displays the accuracy differences between Scikit-learn and PyTorch models after every change, helping users understand performance variations.

Trade Forecasting (Exports & Imports)

Developed a forecasting model for international trade using historical data from the UN Comtrade database. The project applies two approaches — Linear Regression to capture long-term trade trends, and ARIMA (AutoRegressive Integrated Moving Average) to account for time-dependent patterns and seasonal fluctuations. The comparison of both models provides a more robust and reliable forecast of exports and imports for 2025.

Patternova: ML-Driven Kolam Creation Backend

Built an advanced ML backend for Patternova that analyzes and generates intricate Kolam patterns from dot inputs. The system combines traditional dot detection using OpenCV with a synthetic dataset generator for model training, alongside a U-Net image-to-image model (dots → Kolam strokes) implemented in PyTorch. A complete training and inference pipeline ensures efficient processing, while postprocessing converts outputs into skeletonized images and vectorized SVGs. Additional features include data augmentation, automatic grid alignment, and smooth, continuous stroke generation via recursive Bezier-like curves.

API Testing & Documentation
